Debug page

The options in this page indicate the debugging engine settings for a VJET project.

You can reach this page through the Project window (Project > Properties > VJET > Debug) from the context menu on a created project or the File menu of the workbench.

A project can either reuse workspace default settings or use its own custom settings.

Option Description
Enable project specific settings Once selected, debugging engine can be configured for this project as in the Debug preference page. At any time, it is possible to revert to workspace defaults, by using the button Restore Defaults.
