New Source Folder Wizard

The New Source Folder wizard helps you create a new source folder in a VJET project.

An example is shown below.

New Source Folder Options
Option Description Default
Project name Enter a project to contain the new source folder. Either type a valid source folder name or click Browse to select a project via a dialog. The project of the element that was selected when the wizard has been started.
Folder name Type a path for the new source folder. The path is relative to the selected project. <blank>
Update exclusion filter in other source folders to solve nesting Select to modify existing source folder's exclusion filters to solve nesting problems. Not selected

Please refer to the New Source Folder Wizard in Java Development User Guide for detailed introduction of the other options.
