New VJET Package Wizard

The New VJET Package wizard helps you create a folder corresponding to a new VJET package. The corresponding folder of the default package always exists, and therefore doesn't have to be created.

To reach this page please through the File menu (File > New > VJET Package) of the workbench or the Context menu (New > VJET Package) in the Script Explorer.

An example is shown below.

VJET Package Options
Option Description Default
Source folder Type or browse to select a container (project or folder) for the new package. The source folder of the element that was selected when the wizard has been started.
Name Type a name for the new package <blank>

Please refer to the New Java Package Wizard in Java Development User Guide for detailed introduction of the options.
