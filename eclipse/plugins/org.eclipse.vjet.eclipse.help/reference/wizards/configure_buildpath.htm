Configure Build Path

This page help you to specify the build path entries used as default by the new VJET Project creation wizard.
To reach this page please through the Window menu (Window > Preferences > VJET > Build Path) of the workbench.

Configure Build Path Options
Option Description Default
Source folder
Project
The Project folder will be used as root for source files.
Folder
A folder will be created for source files.
Input the name for the folder in the Source folder name textfield.
