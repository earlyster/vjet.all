Editor

The following VJET editor preferences can be set on the VJET > Editor preference page

Note that some options that are generally applicable to text editors can be configured on the text editor preference page.

Option Description Default
Smart caret positioning at line start and end If enabled, the cursor will go the first character of the current row when you type Home On
Smart caret positioning in names If enabled, there are additional word boundaries inside |Camel|Case| Java names. On
Tab policy Select tab character for indentation. Supported space, tab, and mix space and tab characters. Spaces only
