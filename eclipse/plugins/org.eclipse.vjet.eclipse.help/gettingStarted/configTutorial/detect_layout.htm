Detecting existing layout

Layout on file system

  • The source files for a product are laid out in one directory "src".

Steps for defining a corresponding project

  1. Click Window > Open Perspective > Other... > VJET to change to the VJET perspective.

  2. Click File > New > Other... > VJET Project to open the New VJET Project wizard.

  3. Type "Project" in the Project name field.

  4. In Contents group, change selection to Create project from existing source.

    Click Browse... and choose the Product directory on drive C:.

  5. Click Next.

    Ensure that the source folders are detected.

  6. Click Finish.
  7. You now have a VJET project with a "src" folder which contains the sources of the "Project" directory.
