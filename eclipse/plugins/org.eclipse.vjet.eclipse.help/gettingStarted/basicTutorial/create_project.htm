Create a VJET Project

VJET JS features are active within a VJET Project, to create a VJET Project:

  1. Select the menu item File > New > VJET Project to open the New VJET Project wizard.

  2. Enter a project name then click Finish
  3. The newly created VJET Project displays in the Script Explorer. VJET Projects are annotated with the letter 'V' and contain a source folder and the default VJET JS type libraries.
    • src - Source folder
    • Default VJET JS Type Libraries.
      • JS Native Types
      • Browser SDK
      • VJO LIB
